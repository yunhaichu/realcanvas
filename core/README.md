# RealCanvas Core

这里预留给未来共享文档模型、存储契约和平台无关画布算法。

This directory is reserved for the future shared document model, storage
contracts, and platform-neutral canvas algorithms.

Android MVP 可以先使用应用内实现。模型稳定后，再把共享部分抽到这里。

The Android MVP may start with app-local implementations. Once the model
stabilizes, shared pieces can move here.
